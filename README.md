# PRODIGY_ST_03  
## Task 3: Automated Login Test – SauceDemo

### Objective
To automate the login functionality of the SauceDemo website using Selenium with Python, covering both positive and negative test scenarios.

---

### Website Tested
https://www.saucedemo.com/

---

### Test Scenarios Covered
1. Valid Login (standard_user / secret_sauce)
2. Locked Out User
3. Invalid Username
4. Invalid Password
5. Empty Fields

---

### Test Results
- Valid login redirected successfully to the inventory page.
- Locked out user displayed appropriate error message.
- Invalid credentials displayed login error message.
- Empty fields triggered validation error.
- All test cases executed successfully and produced expected outcomes.
